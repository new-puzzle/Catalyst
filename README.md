# Catalyst

This repository contains the source code for Catalyst, a voice-first, privacy-focused productivity companion.

## Project Structure

- **/frontend**: Contains the React (Vite) frontend application.
- **/backend**: Contains the Python (FastAPI) backend application.
- **/data**: Contains local data stores, including the SQLite database and vector memory. This directory is excluded from version control by `.gitignore`.
- **project_requirements.md**: Detailed functional and technical specifications for the project.
- **system_prompt.md**: The core identity and operational instructions for the AI agent.
